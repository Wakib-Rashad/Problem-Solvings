I - Guttu
PDF (English) Ranklist
Time Limit: 2 second(s) Memory Limit: 32 MB

One day Uttug came to Guttu with a strange idea. Uttug asked Guttu to decorate his glass-boxes with colorful marbles. Now Guttu is trying to fill K boxes with colorful marbles. She has an infinite number of colorful marbles. Let us denote the number of marbles in BOXi with Mi ( 1 <= i <= K ). You have to find the number of ways Guttu can fill the K boxes fulfilling the following constraints imposed by Uttug:

·      Each box can contain at most N marbles and must contain at least 1 marble.

·      Mi > Mi-1. for ( 1 < i <= K ).

·      The last digit of Mi-1 and the first digit of Mi should not be the same for ( 1 < i <= K ).

 As the answer can be very large, you have to print the answer modulo 12345.

Input

The first line of input denotes the number of test cases to follow, T( T <= 100 ). Each test case contains two non-negative integers, N(1 <= N <= 200), K( 1 <= K <= 20 ).

Output

For each test case, you have to print a single line. Each line contains the test case number (starting from 1), followed by the answer, the number of ways Guttu can fill the boxes with marbles within the constraints, modulo 12345.

Sample Input

Output for the Sample Input

2

10 1

25 2

 

Case 1: 10

Case 2: 267
