D - I <3 U Q
PDF (English) Ranklist
Time Limit: 2 second(s) Memory Limit: 32 MB

I like the character Q and U very much, especially in this twentieth century. Whenever you open the paper in morning, you will see the news of divorce, or killing even after love marriage. I like Q very much because it almost always goes with U. For example: Queen, Quiet even in Qubee.

In this problem you will be given some English words. You are to output “I <3 U Q” if the word contains at least one “q” which is followed by an “u”. If the word contains one or more “q” but none of them is followed by “u” then output “:(“. Otherwise print “Ordinary Word”.

Input

First line contains number of test case T(<=100). Then there follows T lines each containing a word consisting with only lower case letter. Length of each of the word will not exceed 30.

Output

Print case number and expected output string according to the statement.

Sample Input

Output for the Sample Input

3

mashuq

queen

abc

Case 1: :(

Case 2: I <3 U Q

Case 3: Ordinary Word
