                        Java version of the quickSort and easier one:

                        public void quickSort(int[] arr, int lower, int upper){

                        if(lower < upper) {
                        int i = lower;
                        int j = upper;
                        int pivot = arr[(lower + upper) / 2];
                        while (i < j) {
                        while (arr[i] < pivot)
                        i++;
                        while (arr[j] > pivot)
                        j--;
                        if (i <= j) {
                        int temp = arr[i];
                        arr[i] = arr[j];
                        arr[j] = temp;
                        i++;
                        j--;
                        }
                        }

                        if (lower < j)
                        quickSort(arr, lower, j);

                        if (i < upper)
                        quickSort(arr, i, upper);
                        }
                        }

                                          The idea is, start comparing and swapping first two elements of an array and do same thing again and again with recursion and reach till end. This will push highest value at the end and while coming back from recursion, push lower value to front. This way with one call, we can settle two elements. Now remaining unsorted items are between first and last index therefore we just need to address these unsorted items in similar way.
